# AI-Based Competitive Exam Preparation Assistant

This project is an AI-based platform designed to assist students in preparing for competitive examinations. The system provides mock tests, evaluates user performance, and recommends areas for improvement.

## Features
- User Registration and Login
- Question Bank Management
- Mock Test Generation
- Performance Analysis
- Result Tracking

## Technologies Used
- Python
- Flask
- MySQL
- GitHub
- REST APIs

## Project Structure
backend/
    server.py
frontend/

## Purpose
The goal of this system is to provide an intelligent learning assistant that helps students identify their weak areas and improve their preparation for technical and competitive examinations.
